# DigitalOcean Setup Plan

## Environments
1. Development — dev.realbusinesssuite.com.pg
2. Staging — staging.realbusinesssuite.com.pg
3. Production — app.realbusinesssuite.com.pg

## Recommended First Server
- Basic Droplet
- Ubuntu LTS
- 2GB RAM minimum for development
- PostgreSQL managed database when budget allows

- app.realbusinesssuite.com.pg -> web application
