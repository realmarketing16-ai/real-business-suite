# Sprint 1 Plan — Alpha 0.1

## Objective
Create the first working platform foundation.

## Features
1. User registration and login
2. Company setup wizard
3. Dashboard with KPIs
4. Employee list and profile
5. Departments and branches
6. Roles and permissions foundation
7. Asher AI placeholder panel

## Acceptance Criteria
- User can register and log in.
- User can create a company workspace.
- Company data is separated by company_id.
- User can create, view, update, and archive employees.
- Dashboard shows employee count, leave pending, attendance placeholder, and AI recommendations.
